## Usage guidance

- Use the horizontal logo in website headers and desktop navigation.
- Use the mark for favicons, app icons, social avatars, compact nav, and mobile.
- Use the white variants on dark ink backgrounds.
- Use mono only when one-color reproduction is required.
- Keep clear space around the mark at least equal to the olive dot diameter.
- Do not stretch, rotate, add glow effects, add gradients, or change the olive dot independently.
